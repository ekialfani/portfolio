## Built With
- Tailwindcss
- ReactJS

## Installation
* Clone the repo
  ```sh
  git clone https://github.com/ekialfani/portfolio.git
  ```
* Install dependencies
  ```sh
  npm install
  #or
  yarn install
  ```
* Run the project
  ```sh
  npm start
  #or
  yarn start
  ```
